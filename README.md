# Инструкция по сборке мобильного приложения «Погода+»

Данная инструкция описывает процесс сборки установочного файла (APK) из исходного кода с использованием интерфейса командной строки (CLI) и системы автоматизации сборки Gradle.

## 1. Требования к среде

- Установленный **JDK** (Java Development Kit) версии **11 или выше**.
- Наличие **Gradle Wrapper** (включён в состав проекта — файлы `gradlew` и `gradlew.bat`).
- Переменная окружения `ANDROID_HOME` должна указывать на путь к Android SDK (если сборка производится вне Android Studio).

## 2. Подготовка к сборке

1. Откройте терминал (**Command Prompt**, **PowerShell** или терминал в **Android Studio**).
2. Перейдите в корневую директорию проекта:
   ```bash
   cd путь/к/данному/проекту/WetherApp
## 3. Сборка приложения (CLI)
Для сборки отладочной (debug) версии приложения, которая не требует цифровой подписи для установки, выполните команду:

   ```bash
   gradlew assembleDebug
   ```
## 4. Результат сборки
После завершения процесса (сообщение BUILD SUCCESSFUL) готовый установочный файл будет находиться по следующему пути относительно корня проекта:
```bash
app/build/outputs/apk/debug/app-debug.apk
```

Руководство пользователя приложения «Погода+»
1. Главный экран
При запуске приложения открывается Главный экран. Приложение автоматически подключается к сервису Open-Meteo API и загружает актуальную температуру и скорость ветра для текущего выбранного города (по умолчанию — Москва). Данные отображаются в центральной части экрана и в нижней информационной панели.

2. Раздел «Карта осадков»
Для перехода на карту нажмите кнопку «КАРТА ОСАДКОВ» внизу главного экрана.

Карта реализована с использованием библиотеки OSMDroid (OpenStreetMap).

Карту можно масштабировать двумя пальцами или перемещать свайпами.

В центре карты установлена интерактивная метка целевого города. Нажмите на метку, чтобы увидеть всплывающую подсказку (Snippet). Данные во всплывающей подсказке также формируются динамически на основе актуальных данных из интернета.

3. Раздел «Избранное»
Для перехода к локальной базе данных нажмите «ИЗБРАННОЕ».

В этом разделе вы можете сохранять свои города (данные хранятся локально в SQLite).

Добавление: Введите название в поле и нажмите «Добавить».

Изменение: Нажмите на существующий город в списке, измените его название в поле ввода и нажмите «Изменить».

Удаление: Для удаления города из избранного используйте долгое нажатие на соответствующий элемент списка (или кнопку удаления).
