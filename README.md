Инструкция по сборке мобильного приложения «Погода+»
Данная инструкция описывает процесс сборки установочного файла (APK) из исходного кода с использованием интерфейса командной строки (CLI) и системы автоматизации сборки Gradle.

1. Требования к среде
Установленный JDK (Java Development Kit) версии 11 или выше.

Наличие Gradle Wrapper (включен в состав проекта — файлы gradlew и gradlew.bat).

Переменная окружения ANDROID_HOME должна указывать на путь к Android SDK (если сборка производится вне Android Studio).

2. Подготовка к сборке
Откройте терминал (Command Prompt, PowerShell или терминал в Android Studio).

Перейдите в корневую директорию проекта:

Bash
cd путь/к/вашему/проекту/WeatherApp
3. Сборка приложения (CLI)
Для сборки отладочной (debug) версии приложения, которая не требует цифровой подписи для установки, выполните команду:

Для ОС Windows:

Bash
gradlew assembleDebug
Для ОС macOS / Linux:

Bash
./gradlew assembleDebug
4. Результат сборки
После завершения процесса (сообщение BUILD SUCCESSFUL) готовый установочный файл будет находиться по следующему пути относительно корня проекта:

app/build/outputs/apk/debug/app-debug.apk
